# 模块功能管理

模块一：账号管理
页面内容要包括：
- 账号概览统计卡
  - 已连接账号数
  - 在线账号数
  - 授权即将过期
  - 同步失败账号
- 账号列表表格
  - 账号名称、平台、状态、最近同步时间、操作
- 扫码登录卡片
  - 二维码区域
  - 刷新二维码按钮
  - 扫码状态提示
- Cookie / 授权状态卡片
  - 剩余有效期
  - 重新授权按钮
- 代理配置卡片
  - 代理状态
  - 编辑配置按钮
- 最近同步记录列表
  - 成功 / 失败 / 进行中状态

模块二：知识库
页面内容要包括：
- 知识库概览卡
  - 文档总数
  - 已索引条目
  - 待处理文件
  - 检索命中率
- 文档上传卡片
  - 上传文件按钮
  - 拖拽上传区域
  - 支持格式提示
- 文档列表表格
  - 文件名、类型、大小、状态、更新时间
- 索引进度卡片
  - 当前索引任务
  - 进度条
  - 重新索引按钮
- 检索设置卡片
  - 混合检索开关
  - 权重滑块
  - 保存设置按钮
- 最近查询和命中结果列表
  - 展示搜索词、命中数量、是否命中高质量内容

模块三：内容生成
页面内容要包括：
- 内容草稿概览卡
  - 草稿总数
  - 待审核
  - 已发布
  - 计划发布
- 新建草稿卡片
  - 生成内容按钮
  - 选择账号按钮
  - 选择内容类型
- 草稿列表卡片或表格
  - 标题、账号、状态、更新时间、操作
- 发布计划时间轴
  - 今日待发
  - 明日计划
  - 本周排期
- 模板库卡片
  - 标题模板
  - 封面模板
  - 爆款结构模板
- 最近生成记录列表
  - 成功 / 失败 / 风控拦截状态

模块四：互动管理
页面内容要包括：
- 互动概览卡
  - 待回复评论
  - 待处理私信
  - 已接管会话
  - 超时任务
- 会话列表
  - 左侧会话列表
  - 中间消息线程
  - 右侧详情面板
- 快速回复卡片
  - 常用回复模板
  - 一键发送按钮
- 人工接管控制卡
  - 接管按钮
  - 释放按钮
  - 当前接管状态
- 待处理评论列表
  - 评论内容、来源账号、风险标记、操作
- 实时状态提示
  - 在线 / 离线 / 忙碌

模块五：风控管理
页面内容要包括：
- 风控概览卡
  - 今日扫描数
  - 拦截数
  - 风险评分
  - 通过率
- 内容扫描卡片
  - 输入内容区域
  - 立即扫描按钮
  - 扫描结果区域
- 风险规则列表
  - 敏感词规则
  - 频率限制
  - 竞品过滤
  - 去重规则
- 风险命中列表
  - 规则名称、命中内容、严重级别、处理状态
- 审核动作卡片
  - 通过
  - 拒绝
  - 标记复核
- 风险趋势图或时间轴
  - 最近 7 天风险变化

模块六：数据看板
页面内容要包括：
- 核心指标卡
  - 转化漏斗
  - 内容互动
  - 账号增长
  - LLM 调用统计
- 图表区
  - 折线图
  - 柱状图
  - 漏斗图
- 数据摘要卡
  - 今日新增
  - 本周趋势
  - 本月对比
- 最近活动列表
  - 内容生成
  - 风控拦截
  - 账号同步
  - 审核完成
- 导出报表卡
  - 导出按钮
  - 日期筛选
  - 账号筛选
- 告警概览卡
  - 异常增长
  - 风险异常
  - 同步失败
